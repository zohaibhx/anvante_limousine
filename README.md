# anvante_limousine
Machine Learning Prototype for Real-Time Risk Assessment

📌 Project Overview

This project is an AI-native solution designed to solve a core problem in UAE banking: balancing transaction speed with fraud security. Most models are "black boxes" that either let fraud through or block too many good customers. This project includes a Machine Learning Pipeline to predict risk and a Triage Agent that translates complex AI metrics into actionable business notes for bank stakeholders.

Setup Instructions:
Ensure Python is installed.
Run pip install pandas scikit-learn.
Run python triage_agent.py.
